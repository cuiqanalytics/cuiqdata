Get Started in 3 Simple Steps

Your first pipeline runs in under 2 minutes

Step 1: Download & Install

Download the precompiled binary or installer for your platform from the GitHub Releases page.

# Linux (MacOS version soon)
curl -L https://github.com/cuiqanalytics/cuiqdata/releases/download/latest/cuiqdata -o cuiqdata
chmod +x cuiqdata
./cuiqdata demo

# Windows (download installer)
# https://github.com/cuiqanalytics/cuiqdata/releases/download/v0.3.1/cuiqdata-windows-x64.exe
# Then run: cuiqdata demo

Step 2: Run Your First Pipeline

Navigate to the demo project and execute the sample pipeline.

cd demo
cuiqdata run sql

View execution summary, cached steps, and output data instantly.
