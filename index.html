Get Started in 3 Simple Steps

Your first pipeline runs in under 2 minutes

Step 1: Download & Install

Download the precompiled binary or installer for your platform from the GitHub Releases page.

# On Linux (MacOS version soon)
curl -L https://github.com/cuiqanalytics/cuiqdata/releases/download/v0.3.1/cuiqdata-linux-x86-64 -o cuiqdata
chmod +x cuiqdata

Step 2: Run Your First Pipeline

Open a terminal Navigate to the demo project and execute the sample pipeline.

./cuiqdata demo
cd demo
./cuiqdata run sql

Step 3: See the Results

View execution summary, cached steps, and output data instantly.
